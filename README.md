# Customer-Behavior-Analytics-PowerBI

## 📌 Overview
This project analyzes customer purchasing behavior using an end-to-end data analytics workflow.

The goal was to understand:
- revenue patterns
- subscription trends
- customer segmentation
- category and shipping performance

---

## 🛠️ Tools & Technologies
- Python (Jupyter Notebook) → Data Cleaning
- MySQL Workbench → Business Analysis
- Power BI → Dashboard & Visualization
- DAX → KPI Calculations

---

## 🔄 Workflow

1. Data Cleaning (Jupyter Notebook)
   - handled missing values
   - formatted columns
   - prepared dataset for analysis

2. SQL Analysis (MySQL)
   - analyzed customer trends
   - category performance queries
   - repeat customer analysis

3. Dashboard (Power BI)
   - built KPI cards (Revenue, Customers, Avg Purchase)
   - created interactive visuals
   - added slicers for filtering

---

## 📊 Key Insights
- Total Revenue: 233K
- Subscription Rate: 27%
- Highest Revenue Category: Clothing
- Male customers generated higher revenue than female customers
- Free Shipping is the most used shipping method
